# Perceptron Placement Prediction

This project demonstrates the implementation of the Perceptron algorithm using Scikit-Learn.

Dataset Features:
- CGPA
- Resume Score

Target:
- Placement Status (Placed / Not Placed)

The project covers:
- Data visualization using Seaborn
- Training a Perceptron classifier
- Understanding model coefficients and bias
- Visualizing decision boundaries using mlxtend

Technologies Used:
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- mlxtend
